# KAFFA Café

Proyecto de frontend hecho con React + Vite + Tailwind.

## Cómo ejecutar

1. Instala las dependencias:

```bash
npm install
```

2. Inicia el servidor local:

```bash
npm run dev
```

3. Abre el navegador en:

```bash
http://localhost:5173
```

## Scripts disponibles

- `npm run dev` - Ejecuta la app en modo desarrollo con Vite.
- `npm run build` - Genera la versión de producción en `dist`.
- `npm run preview` - Previsualiza la app de producción.

## Notas

- El proyecto ya no usa Create React App.
- Esta versión funciona con Vite y Tailwind.
- Si quieres ver la app, usa la URL que te muestra Vite en la terminal.
